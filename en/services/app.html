  • Natives

    It means that for its development the native programming language of the device, Objective C or Swift for iOS, Java for Android and .Net for Windows Phone. Is a hundred percent model dependent on the platform and the Apps are not portable, you have to develop one per platform. — Some native application examples would be Whatsapp or Facebook.

  • Hybrid

    They are applications developed using HTML5, CSS and JavaScript, deployed within a native container like Phonegap / Cordova which provides access to the capabilities of the device in a completely neutral way with respect to the operating system. It is a neutral model with respect to the platform and with maximum portability.

  • Generated

    These are applications developed using tools such as Xamarin or Genexus (between many others), where the development is carried out using specific techniques and languages of the tool and then the App is generated in the language of the target platform to be compiled with the native tools.
